# Features & Changes

### Features
* Lots of Magic and Tech Mods
* Several Unknown/Uncommon Addons
* Unique Ore Manufacturing
* Liesurely Skyblock Experience
* Recipe Changes
* Forced Progression
* Extreme Crafting for End-game Items
